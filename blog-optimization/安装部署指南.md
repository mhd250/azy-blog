# ============================================
# Hexo博客优化完整安装部署指南
# 适用于安知鱼主题
# ============================================

## 📋 前置要求

- Node.js >= 14.0
- Git
- Python 3.7+（用于一键发布脚本）

## 🚀 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/mhd250/azy-blog.git
cd azy-blog
```

### 2. 安装依赖

```bash
# 安装Hexo依赖
npm install

# 安装主题（如果themes目录为空）
npm install hexo-theme-anzhiyu

# 安装优化插件
npm install hexo-generator-search
npm install hexo-generator-feed
npm install hexo-generator-sitemap
npm install hexo-all-minifier
```

### 3. 应用优化配置

#### 3.1 复制苹果风格CSS

```bash
# 创建CSS目录
mkdir -p source/css

# 复制CSS文件
cp blog-optimization/apple-glassmorphism.css source/css/
```

#### 3.2 修改主题配置

编辑 `_config.anzhiyu.yml`，添加：

```yaml
# 引入自定义CSS
inject:
  head:
    - <link rel="stylesheet" href="/css/apple-glassmorphism.css">
```

#### 3.3 配置搜索功能

编辑 `_config.yml`，添加：

```yaml
search:
  path: search.xml
  field: post
  content: true
```

编辑 `_config.anzhiyu.yml`，启用搜索：

```yaml
local_search:
  enable: true
  preload: true
  top_n_per_article: 3
```

### 4. 安装一键发布脚本

```bash
# 创建tools目录
mkdir -p tools

# 复制脚本
cp blog-optimization/publish.py tools/

# 安装Python依赖
pip install Pillow

# 添加执行权限（Linux/Mac）
chmod +x tools/publish.py
```

### 5. 本地预览

```bash
hexo clean
hexo generate
hexo server
```

访问 http://localhost:4000/azy-blog/

### 6. 部署到GitHub Pages

```bash
hexo deploy
```

**文件**: `source/css/apple-glassmorphism.css`

**效果**:
- 导航栏毛玻璃效果 ✅
- 文章卡片毛玻璃效果 ✅
- 侧边栏毛玻璃效果 ✅

**自定义**:
```css
:root {
  --apple-blur-bg: rgba(255, 255, 255, 0.72);  /* 调整透明度 */
  --apple-backdrop-filter: saturate(180%) blur(20px);  /* 调整模糊度 */
}
```

### 配置2: 搜索功能

**安装**:
```bash
npm install hexo-generator-search
```

**配置**:
```yaml
# _config.yml
search:
  path: search.xml
  field: post
  content: true
```

### 配置3: RSS订阅

**安装**:
```bash
npm install hexo-generator-feed
```

**配置**:
```yaml
# _config.yml
feed:
  enable: true
  type:
    - atom
    - rss2
  path:
    - atom.xml
    - rss2.xml
  limit: 20
```

### 配置4: 站点地图

**安装**:
```bash
npm install hexo-generator-sitemap
npm install hexo-generator-baidu-sitemap
```

**配置**:
```yaml
# _config.yml
sitemap:
  path: sitemap.xml

baidusitemap:
  path: baidusitemap.xml
```

### 配置5: 图片懒加载

**安装**:
```bash
npm install hexo-lazyload-image
```

**配置**:
```yaml
# _config.yml
lazyload:
  enable: true
  onlypost: false
  loadingImg: /img/loading.gif
```

## 📝 使用一键发布脚本

### 基本用法

```bash
# 创建普通文章
python tools/publish.py -t "我的文章标题"

# 创建技术文章
python tools/publish.py -t "Vue3教程" --type tech -c "技术" --tags "vue,前端"

# 创建置顶文章
python tools/publish.py -t "重要公告" --top

# 创建并立即部署
python tools/publish.py -t "新文章" --deploy

# 列出所有文章
python tools/publish.py --list
```

### 文章类型

- `article`: 普通文章
- `tech`: 技术文章（带代码模板）
- `life`: 生活文章（带图片模板）
- `tutorial`: 教程文章（带步骤模板）

### 自动生成的Front-matter


## 🎨 自定义主题

### 修改配色

编辑 `source/css/apple-glassmorphism.css`:

```css
:root {
  /* 浅色模式 */
  --apple-blur-bg: rgba(255, 255, 255, 0.72);

  /* 深色模式 */
  --apple-blur-bg-dark: rgba(28, 28, 30, 0.72);
}
```

### 修改字体

在 `_config.anzhiyu.yml` 中：

```yaml
font:
  global:
    enable: true
    family: "Noto Sans SC"
    size: 16px
  code:
    enable: true
    family: "JetBrains Mono"
    size: 14px
```

### 添加自定义JS

创建 `source/js/custom.js`:

```javascript
// 你的自定义代码
console.log('Hello, World!');
```

在 `_config.anzhiyu.yml` 中引入：

```yaml
inject:
  bottom:
    - <script src="/js/custom.js"></script>
```

## 🐛 常见问题

### Q1: 搜索功能不工作

**解决**:
1. 检查 `search.xml` 是否生成
2. 确认 `hexo-generator-search` 已安装
3. 检查主题配置中的搜索设置

### Q2: 毛玻璃效果不显示

**解决**:
1. 检查CSS文件是否正确引入
2. 确认浏览器支持 `backdrop-filter`
3. 检查是否有其他CSS覆盖

### Q3: 部署后样式丢失

**解决**:
1. 运行 `hexo clean`
2. 重新生成 `hexo generate`
3. 检查GitHub Pages设置

### Q4: 图片加载慢

**解决**:
1. 启用懒加载
2. 压缩图片大小
3. 使用CDN加速
4. 使用WebP格式
