## 方案一：本地搜索（推荐）

### 1. 安装搜索插件

```bash
npm install hexo-generator-search --save
```

### 2. 配置搜索

在 `_config.yml` 中添加：

```yaml
search:
  path: search.xml
  field: post
  content: true
  template: ./search_template.xml
```

### 3. 安知鱼主题搜索配置

在 `_config.anzhiyu.yml` 中配置：

```yaml
# 本地搜索配置
local_search:
  enable: true
  preload: true  # 预加载搜索数据
  top_n_per_article: 3  # 每篇文章显示的搜索结果数
  unescape: false
  CDN: false

# 搜索框样式
search:
  enable: true
  type: local  # local/algolia
  placeholder: "搜索文章..."
  format: "$title$ - $content$"
  limit: 10
```

### 5. 搜索功能增强脚本

创建 `source/js/search-enhance.js`：

```javascript
/**
 * 搜索功能增强
 * 添加快捷键、搜索历史、热门搜索等功能
 */

(function() {
  'use strict';

  // 配置
  const config = {
    shortcutKey: '/',  // 打开搜索的快捷键
    maxHistory: 5,     // 最大历史记录数
    enableHistory: true,
    enableHot: true
  };

  // 搜索历史管理
  const SearchHistory = {
    key: 'azy_search_history',

    get() {
      try {
        return JSON.parse(localStorage.getItem(this.key)) || [];
      } catch {
        return [];
      }
    },

    save(keyword) {
      if (!keyword.trim()) return;
      let history = this.get();
      history = history.filter(item => item !== keyword);
      history.unshift(keyword);
      history = history.slice(0, config.maxHistory);
      localStorage.setItem(this.key, JSON.stringify(history));
    },

    clear() {
      localStorage.removeItem(this.key);
    },

    render() {
      const history = this.get();
      const container = document.querySelector('.search-history');
      if (!container || history.length === 0) return;

      container.innerHTML = `
        <div class="search-history-title">搜索历史</div>
        ${history.map(item => `
          <span class="search-history-item" data-keyword="${item}">${item}</span>
        `).join('')}
      `;

      // 绑定点击事件
      container.querySelectorAll('.search-history-item').forEach(item => {
        item.addEventListener('click', () => {
          const keyword = item.dataset.keyword;
          const input = document.querySelector('#local-search .search-input');
          if (input) {
            input.value = keyword;
            input.dispatchEvent(new Event('input'));
          }
        });
      });
    }
  };

  // 热门搜索（可以动态获取或静态配置）
  const HotSearch = {
    keywords: ['Hexo', '前端', 'Vue', 'React', 'JavaScript', 'CSS'],

    render() {
      const container = document.querySelector('.search-hot');
      if (!container) return;

      container.innerHTML = `
        <div class="search-hot-title">热门搜索</div>
        ${this.keywords.map(item => `
          <span class="search-hot-item" data-keyword="${item}">${item}</span>
        `).join('')}
      `;

      // 绑定点击事件
      container.querySelectorAll('.search-hot-item').forEach(item => {
        item.addEventListener('click', () => {
          const keyword = item.dataset.keyword;
          const input = document.querySelector('#local-search .search-input');
          if (input) {
            input.value = keyword;
            input.dispatchEvent(new Event('input'));
          }
        });
      });
    }
  };

  // 快捷键支持
  function initShortcut() {
    document.addEventListener('keydown', (e) => {
      // 按 / 打开搜索（不在输入框中时）
      if (e.key === config.shortcutKey && 
          e.target.tagName !== 'INPUT' && 
          e.target.tagName !== 'TEXTAREA') {
        e.preventDefault();
        const searchBtn = document.querySelector('.nav-search-btn');
        if (searchBtn) searchBtn.click();
      }

      // 按 ESC 关闭搜索
      if (e.key === 'Escape') {
        const closeBtn = document.querySelector('.search-close');
        if (closeBtn) closeBtn.click();
      }
    });
  }

  // 保存搜索历史
  function initSearchHistory() {
    const input = document.querySelector('#local-search .search-input');
    if (input) {
      input.addEventListener('change', (e) => {
        SearchHistory.save(e.target.value);
      });
    }
  }

  // 初始化
  function init() {
    // 等待搜索框加载完成
    const checkExist = setInterval(() => {
      const searchDialog = document.querySelector('#local-search .search-dialog');
      if (searchDialog) {
        clearInterval(checkExist);

        // 添加搜索历史和热门搜索容器
        const resultList = searchDialog.querySelector('.search-result-list');
        if (resultList) {
          // 在结果列表前插入历史和热门
          const historyDiv = document.createElement('div');
          historyDiv.className = 'search-history';
          resultList.parentNode.insertBefore(historyDiv, resultList);

          const hotDiv = document.createElement('div');
          hotDiv.className = 'search-hot';
          resultList.parentNode.insertBefore(hotDiv, resultList);

          // 渲染
          SearchHistory.render();
          HotSearch.render();
        }

        // 添加快捷键提示
        const closeBtn = searchDialog.querySelector('.search-close');
        if (closeBtn) {
          const shortcutTip = document.createElement('span');
          shortcutTip.className = 'search-shortcut-tip';
          shortcutTip.textContent = '按 ESC 关闭';
          closeBtn.parentNode.appendChild(shortcutTip);
        }

        initSearchHistory();
      }
    }, 100);

    initShortcut();
  }

  // 页面加载完成后初始化
  if (document.readyState === 'loading') {
    document.addEventListener('DOMContentLoaded', init);
  } else {
    init();
  }
})();
```

## 方案二：Algolia搜索（高级）

### 1. 注册Algolia

1. 访问 [Algolia](https://www.algolia.com/)
2. 注册账号并创建应用
3. 获取 Application ID 和 Search-Only API Key

### 2. 安装插件

```bash
npm install hexo-algolia --save
```

### 3. 配置 Algolia

在 `_config.yml` 中添加：

```yaml
algolia:
  applicationID: '你的Application ID'
  apiKey: '你的Search-Only API Key'
  indexName: 'azy-blog'
  chunkSize: 5000
  fields:
    - title
    - content
    - categories
    - tags
    - permalink
    - date
```

### 4. 主题配置

```yaml
search:
  enable: true
  type: algolia
  placeholder: "搜索文章..."
```

### 5. 推送数据到Algolia

```bash
hexo algolia
```

## 方案对比

| 特性 | 本地搜索 | Algolia |
|------|---------|---------|
| 成本 | 免费 | 免费额度内 |
| 速度 | 快 | 非常快 |
| 准确性 | 良好 | 优秀 |
| 实时性 | 需重新生成 | 实时 |
| 配置难度 | 简单 | 中等 |
| 自定义程度 | 高 | 中等 |
| 适合规模 | <1000篇文章 | 任意规模 |

**推荐**: 对于个人博客，本地搜索已足够，配置简单且完全免费。

## 常见问题

### Q1: 搜索无结果
- 检查 `search.xml` 是否生成
- 确认搜索配置正确
- 检查文件路径

### Q2: 搜索速度慢
- 减少每篇文章显示的结果数
- 优化搜索数据大小
- 考虑使用Algolia

### Q3: 中文搜索不准确
- 使用更智能的搜索插件
- 配置中文分词
- 考虑使用Algolia的中文支持

---

## 优化建议

1. **定期更新搜索数据**: 每次发布文章后重新生成
2. **优化文章内容**: 提高关键词密度
3. **添加标签和分类**: 提高搜索准确性
4. **使用描述**: 为每篇文章添加description
5. **监控搜索日志**: 了解用户搜索习惯
