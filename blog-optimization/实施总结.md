## 🚀 实施步骤

### 第一步：应用优化（5分钟）

```bash
# 1. 进入博客目录
cd azy-blog

# 2. 复制优化文件到博客目录
cp -r /path/to/blog-optimization/* .

# 3. 运行一键配置脚本
python setup.py
```

### 第二步：配置搜索（2分钟）

编辑 `_config.anzhiyu.yml`:
```yaml
local_search:
  enable: true
  preload: true
  top_n_per_article: 3
```

### 第三步：部署（2分钟）

```bash
hexo clean
hexo generate
hexo deploy
```

## 🎨 自定义建议

### 修改配色

编辑 `source/css/apple-style.css`:
```css
:root {
  --apple-blur-bg: rgba(255, 255, 255, 0.72);
  --apple-backdrop-filter: saturate(180%) blur(20px);
}
```

### 修改字体

在 `_config.anzhiyu.yml`:
```yaml
font:
  global:
    family: "Noto Sans SC"
    size: 16px
```

## 🐛 常见问题

### Q1: 毛玻璃效果不显示
**解决**: 使用最新版Chrome/Safari浏览器

### Q2: 搜索功能不工作
**解决**: 运行 `hexo generate` 生成搜索数据

### Q3: 部署后样式丢失
**解决**: 运行 `hexo clean` 后重新部署

### Q4: 如何恢复原始样式
**解决**: 删除 `source/css/apple-style.css` 并移除配置中的引用
