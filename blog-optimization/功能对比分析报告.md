## 三、推荐新增功能清单

### 🔴 高优先级（建议立即添加）

#### 1. 文章搜索功能
**重要性**: ⭐⭐⭐⭐⭐
**实现难度**: ⭐⭐
**预计时间**: 30分钟

**实现方案**:
```bash
npm install hexo-generator-search
```

**配置** (_config.anzhiyu.yml):
```yaml
local_search:
  enable: true
  preload: true
  top_n_per_article: 3
  unescape: false
  CDN: false
```

#### 2. 图片懒加载优化
**重要性**: ⭐⭐⭐⭐⭐
**实现难度**: ⭐
**预计时间**: 10分钟

**配置**:
```yaml
lazyload:
  enable: true
  field: site
  placeholder: /img/loading.gif
  blur: true
```

#### 3. PWA支持
**重要性**: ⭐⭐⭐⭐
**实现难度**: ⭐⭐⭐
**预计时间**: 1小时

**实现步骤**:
1. 安装PWA插件
2. 配置manifest.json
3. 添加Service Worker
4. 配置图标

#### 4. RSS订阅
**重要性**: ⭐⭐⭐⭐
**实现难度**: ⭐
**预计时间**: 10分钟
